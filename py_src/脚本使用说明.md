# 脚本使用说明

## 安装运行环境

1. 参考《python环境搭建.md》安装python
2. 安装程序运行需要的软件包：`pip3 install requests beautifulsoup4 lxml`

## 运行Py文件

```
python3 500.py

# 支持指定时间范围，注意：参数要加双引号
# 指定起始日期
python3 500.py --range "2026-01-10"
# 指定起始日期和结束日期，可省略小时和分钟
python3 500.py --range "2026-01-10~2026-01-10 23:59"
# 指定起始日期和结束日期，指定完整的日期时间
python3 500.py --range "2026-01-10 00:00~2026-01-10 23:59"
# 指定起始日期和结束日期，指定完整的日期时间，跨天指定
python3 500.py --range "2026-01-10 00:00~2026-01-11 04:00"
# 指定起始日期和结束日期，指定完整的日期时间，起始与结束相同
python3 500.py --range "2026-01-11 04:00~2026-01-11 04:00"
```
