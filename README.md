# Football Betting

一、需要准备这 4 大类数据（核心）
====================

✅ ① 比赛基础信息（必需，缺一不可）
-------------------

这是“坐标系”，不是分析点。

*   联赛 / 杯赛名称
*   比赛时间（精确到北京时间即可）
*   主队 vs 客队

📌 示例格式：

```
英超
富勒姆 vs 诺丁汉森林
北京时间 2025-12-23 04:00
```

* * *

✅ ② 【初盘】关键数据（最重要的一组）
--------------------

这是是否能做出“方向判断”的根基。

只需要 **2 家**，而且必须是：

**必须提供**

*   **Pinnacle（平博）初盘**
*   **皇冠 初盘**

👉 玩法按要分析的来：

*   胜平负 / 让球胜平负
*   或 亚盘
*   或 大小球

📌 示例（亚盘）：

```
Pinnacle 初盘：
主 -0.5  @0.96

皇冠 初盘：
主 -0.5  @0.94
```

⚠️ 关键提醒：

*   **一定要是“初盘”**，不是即时
*   哪怕是在 500网站 看到的“最早一条记录”，也可以

* * *

✅ ③ 【早期 & 中期】变化数据（判断结构用）
------------------------

这里**给“代表性变化”就够**。

**推荐提供这几家中的任意 3–5 家：**

*   威廉希尔
*   立博
*   澳门
*   易胜博
*   Bet365

给 **变化轨迹**，而不是单一时刻。

📌 示例（胜平负）：

```
威廉希尔：
主胜 1.90 → 1.78

立博：
主胜 1.88 → 1.82

Bet365：
主胜 1.85 → 1.72
```

AI需要关心的是：

*   谁先动
*   动的是赔率还是盘口
*   是主动偏离，还是被动跟随

* * *

✅ ④ 【临场】验证数据（可选，但加分）
--------------------

这部分可以做**风控判断**。

两种任选其一即可：

**A. Bet365 临场价格**

```
Bet365 临场：
主胜 1.65
```

**B. 必发（Betfair）简要信息**

不用给全表，只要：

*   主 / 客 各自成交比例（大概）
*   是否有明显异常价位

📌 示例：

```
必发：
主胜成交量约 70%
价格持续下压
```

* * *

二、【不需要】提供的东西（刻意省掉）
===================

这点很重要，要帮你去掉噪音。

❌ 不需要：

*   技战术分析
*   阵容预测
*   伤停新闻
*   媒体观点
*   谁克制谁
*   你个人的感觉

👉 **这些只会在已经“选错阶段庄家”时，帮你合理化错误。**

* * *

三、照这个模板，就能直接实战
=================

以后可以直接复制下面这个「标准输入模板」👇

```
【比赛】
联赛：
时间：
对阵：

【初盘】
Pinnacle：
皇冠：

【变化】
威廉 / 立博 / 澳门 / 易胜博 / Bet365（任选3–5）：

【临场（可选）】
Bet365：
或
必发：
```

给到这里，会明确告诉你：

*   这场 **能不能碰**
*   如果能碰，**风险在哪一侧**
*   如果不该碰，**是哪一个阶段已经出问题**

* * *

四、非常重要的一句实话
=============

> **真正决定胜负的，不是你“会不会分析”，  
> 而是你有没有在“错误阶段，用错误的数据”。**

现在做的这一步（明确输入标准），  
已经是**90% 的玩家永远不会做的事**。
